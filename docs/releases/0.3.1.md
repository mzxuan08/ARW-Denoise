# ARW Denoise 0.3.1

## 交付摘要

- 设置页可选择缓存存放位置，适合将预览缓存放到 F 盘等非系统盘。
- 自定义位置始终使用独立的 `ArwDenoiseCache` 子目录。
- 切换缓存位置后立即生效，并自动清理旧预览缓存。
- 新增“打开缓存目录”和“一键清理缓存”，清理前显示文件数和占用空间。
- 缓存目录使用专用标记校验；不会删除同级 ARW、DNG 或其他用户文件。
- 保持 0.3.0 的完整离线 PMRID / CUDA、批处理、可编辑 DNG、进度、取消和对比预览能力。

## 兼容性

- 设置格式向后兼容 0.3.0；未配置缓存位置时继续使用 Windows 用户数据目录。
- DNG 中间文件仍与最终导出目录同盘，以维持原子发布和取消清理保证。
- 正式目标平台为 Windows 10/11 x64、Sony A7C II 和 NVIDIA RTX 3070 Ti Laptop。
