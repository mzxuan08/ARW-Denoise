# ARW Denoise 0.5.1

## 紧急修复

- 修复 GUI 工作线程以位置参数构造 `ProgressTracker`，导致所有队列任务在解码前立即失败的问题。
- GUI 现在通过共用、可测试的进度控制器入口传递任务 ID。
- 增加回归测试，验证 GUI 任务能够产生带正确任务 ID 的解码进度。
- 降噪算法、PMRID 模型、自动参数、DNG 像素写入和队列数据库结构均未改变。

## 0.5.0 失败任务

原队列不会丢失。安装 0.5.1 后点击“重试失败任务”，再点击“开始处理”即可。
