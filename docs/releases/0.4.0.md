# ARW Denoise 0.4.0

## 交付摘要

- 移除 Sony A7C II（ILCE-7CM2）型号白名单。
- 所有 Sony 型号统一按实际 RAW 能力检测，不需要软件预先认识相机型号。
- 覆盖 A1、A9、A7、A6xxx、A5xxx、NEX、FX、ZV、RX、SLT、DSLR 等 Sony ARW 产品线及未来型号。
- 继续保留相机 Make/Model、2×2 Bayer CFA、黑白电平、活动区域、白平衡、色彩矩阵和高光余量。
- 非 Sony 文件、非 2×2 Bayer CFA、无效位深、错误活动区域或矛盾电平会明确拒绝，不静默猜测。
- 包含 0.3.1 的非系统盘缓存位置、打开缓存目录和安全一键清理功能。

## 兼容性原则

软件接受满足以下条件的 Sony ARW：

1. LibRaw/rawpy 能读取原始 CFA 像素；
2. dnglab 能读取并保留 RAW/DNG 元数据；
3. 可见区域为有效的偶数尺寸 2×2 Bayer 阵列；
4. CFA 含一个红色、两个绿色和一个蓝色采样；
5. 位深为 8–16 bit，且白电平高于四通道黑电平。

当前实机性能与 DNG 回归样本仍来自 Sony A7C II / RTX 3070 Ti Laptop。其他 Sony 型号无需代码白名单，但首次投入批量生产前仍建议抽样在 Adobe Camera Raw、Lightroom 或像素蛋糕中人工验收。
