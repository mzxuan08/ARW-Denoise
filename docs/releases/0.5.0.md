# ARW Denoise 0.5.0

## 交付摘要

- ARW 文件和文件夹在进入队列前执行后台兼容性预检，GUI 不因大量文件扫描而冻结。
- 文件夹扫描默认递归，自动发现所有子目录中的 `.ARW` / `.arw`。
- 预检采用两个独立解码器并发执行，适合相机卡和机械硬盘，不用过高并发争抢磁盘。
- 只有通过 Sony 品牌、2×2 Bayer、位深、黑白电平、活动区域检查的文件才进入队列。
- 队列持久化相机 Make/Model、ISO、可见尺寸、位深和 CFA 信息，并直接显示关键字段。
- 不兼容文件不会阻断其他照片；GUI 展示错误摘要并保存 `compatibility-report-latest.json`。
- 新增 CLI `scan`，支持多文件、多目录、递归扫描、1–8 个 worker 和 UTF-8 JSON 报告。
- ZIP 改用平衡压缩，ZIP/7z 默认限制为 4 个线程，减少发布时对日常操作的影响；最小体积继续使用固实 LZMA2 7z。

## 真实批量扫描

在 `E:\ych\01` 的 175 张 Sony A7C II ARW（共约 6.72 GB）上执行：

- 总数：175
- 兼容：175
- 不兼容：0
- 报告：`outputs/COMPATIBILITY-SCAN-0.5.0.json`

兼容性预检不会修改源 ARW，也不会创建 DNG。
