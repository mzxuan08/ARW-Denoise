# ARW Denoise 0.3.0

## 交付摘要

- 当前照片与整队列进度、阶段、已用时间和 ETA。
- 立即取消当前 tile 或 dnglab 子进程，不发布半成品 DNG。
- GPU 会话/输入缓冲复用，单线程预解码下一张 RAW，GPU 仍串行处理。
- 完全离线的源 ARW / 降噪 DNG 对比窗口：适应、100%、同步缩放/平移和可拖分割线。
- 分开“重试失败”与“重试已取消”；清理已完成记录不删除 DNG。
- 受控发布清单、SHA-256、兼容 ZIP 和固实 LZMA2 双格式交付。

## A7C II / RTX 3070 Ti Laptop 实测

- 0.2.0 基线：6 个正式样本中位总耗时 5.600 秒/张，RAM 峰值 1,855–2,246 MB。
- 0.3.0 预取队列：6 个正式样本实际墙钟吞吐 3.918 秒/张，提升约 30.0%。
- 0.3.0 RAM 峰值约 1,573–1,729 MB；相对基线最高值降低约 29.9%。
- Windows WDDM 不报告可用的逐进程显存，记录的 1,414 MB 是测试时整卡占用峰值。
- 两张 A7C II 验收 DNG 与 0.2.0 基线各 32,966,016 个 CFA 采样逐像素对比：最大差值 0，不同采样数 0。
- 半尺寸对比预览冷生成约 1.55 秒，缓存命中约 0.03 秒。

## 归档体积

- ZIP：1,783,370,562 字节。
- 固实 7z：1,179,211,004 字节，比 ZIP 小 33.9%。

源 ARW 始终只读。验收 DNG 应继续在 Adobe Camera Raw/Lightroom 或像素蛋糕的实际工作流中做人工视觉确认。
